# Real-Time Object Detection using YOLOv8

This project implements a real-time object detection system using the YOLOv8 Nano model. It captures live video input from a webcam and performs object detection with bounding boxes and class labels.

---

## Overview

The objective of this project was to gain hands-on experience with object detection systems and understand how real-time inference works in practice.

The implementation focuses on processing live video streams, running model inference, and visualizing detection results using bounding boxes.

---

## Features

- Real-time object detection using webcam input  
- YOLOv8 Nano model for fast inference  
- Bounding box visualization with class labels  
- Lightweight and efficient implementation  

---

## Tech Stack

- Python  
- OpenCV  
- Ultralytics YOLOv8
